Acerca de mí

Mi nombre es Matías Ghigi y este proyecto forma parte de la Entrega Final del curso de Python en Coderhouse.

El objetivo de esta aplicación es poner en práctica los conceptos principales de desarrollo web con Django, creando una plataforma estilo blog con gestión de usuarios, perfiles personalizados, páginas de contenido y mensajería interna.

A lo largo del proyecto se trabajaron temas como:

  • Arquitectura de proyectos Django
  • Autenticación y autorización de usuarios
  • CRUD completo de modelos
  • Herencia de templates
  • Uso de Class Based Views y decoradores
  • Carga de imágenes y texto enriquecido

La aplicación está pensada como una base sólida y escalable, sobre la cual se pueden seguir agregando funcionalidades y mejoras de diseño.

Gracias por visitar el proyecto.
